# 📝 Travel Journal (여행 일기)

사용자가 여행의 추억을 기록하고 관리할 수 있는 웹 애플리케이션. 브라우저의 `localStorage`를 활용하여 오프라인 상태에서도 모든 기능이 정상적으로 작동.

!Travel Journal Screenshot


✨ **주요 기능**
- **여행 기록 생성:** 도시, 날짜, 메모, 사진을 포함한 새 기록 추가 기능
- **오프라인 저장:** 모든 데이터를 브라우저 `localStorage`에 저장하여 인터넷 연결 없이도 사용 가능
- **사진 첨부:** 각 기록에 사진 한 장을 첨부하고 함께 저장하는 기능
- **기록 관리:** 생성된 여행 기록을 삭제하는 기능
- **사용자 친화적 인터페이스:** 파일 첨부 상태 표시 등 직관적인 UI/UX 제공

🛠️ **기술 스택**
- **Frontend:** HTML, CSS, Vanilla JavaScript
- **Storage:** Browser `localStorage` API
